Anaconda报错：No module named 'urllib3.packages.six.moves'

约 224 字小于 1 分钟

Anaconda报错

2025-09-17

报错提示

在启动conda时报错：ModuleNotFoundError: No module named 'urllib3.packages.six.moves'

解决方法

参照网上方法执行下面指令：

pip3 uninstall urllib3 -y --cert root.pem

pip3 install --no-cache-dir -U urllib3

成功启动conda，但是存在告警。
